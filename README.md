# Books

Portafolio profesional de Victor Ortuno.

Dominio previsto: `books.merkana.club`

## Comandos

```bash
npm.cmd install
npm.cmd run dev
npm.cmd run build
```

## Publicacion

El sitio se publica con GitHub Pages desde `.github/workflows/deploy-pages.yml`.

Para que el dominio funcione, el DNS de `books.merkana.club` debe apuntar a GitHub Pages.
